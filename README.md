# 🛒 Lista de Compras

Aplicativo desenvolvido em **Flutter** para gerenciar uma **lista de compras** de forma simples e prática.

## 🚀 Funcionalidades

- Adicionar novos produtos à lista
- Excluir produtos
- Marcar produtos como comprados

## 🛠️ Tecnologias utilizadas

- Flutter
- Dart
- Material Design
